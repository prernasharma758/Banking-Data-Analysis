# Banking-Data-Analysis
Banking dataset analysis using Python and Pandas

## 📌 Overview
This project analyzes banking data to understand customer behavior and transaction patterns.

## 🎯 Objective
- Perform data cleaning
- Analyze customer transactions
- Generate insights

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib

## 📊 Key Insights
- High balance customers make more transactions
- Some age groups are more active

## 🚀 Conclusion
Data analysis helps banks make better decisions.
